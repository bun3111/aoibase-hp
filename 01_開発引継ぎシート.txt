■ デプロイ手順
================================================================

1. ローカルでテーマファイルを編集
2. Python zipfile で ZIP 作成（フォワードスラッシュ必須）
   Python パス: /c/Users/takut/AppData/Local/Programs/Python/Python312/python
3. WordPress 管理画面 > 外観 > テーマ
4. Twenty Twenty-Five に切替
5. aoibase-theme を削除
6. 新 ZIP をアップロード → インストール → 有効化

※ アクティブテーマのまま上書きアップロードは不可
※ python3 コマンドは Windows Store スタブのため使用不可

■ 注意事項
================================================================

- ZIP 作成時は必ず Python zipfile を使用（PowerShell 不可）
- 画像ファイル名は ASCII のみ使用（日本語名は Linux で問題を起こす）
- テーマ更新時は必ず別テーマに切替→削除→再アップロードの手順
- CF7フォームはWordPress管理画面で設定済み（テーマファイルにはショートコードのみ）
- main.js はスライド数を動的取得（slides.length）するため、HTML側のスライド増減だけでOK
