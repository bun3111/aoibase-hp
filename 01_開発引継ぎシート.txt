■ デプロイ手順
================================================================

1. ローカルでテーマファイルを編集
2. Python zipfile で ZIP 作成
   - Windows: /c/Users/takut/AppData/Local/Programs/Python/Python312/python
   - Mac: python3 が使用可能
3. WordPress 管理画面 > 外観 > テーマ
4. Twenty Twenty-Five に切替
5. aoibase-theme を削除
6. 新 ZIP をアップロード → インストール → 有効化

※ アクティブテーマのまま上書きアップロードは不可
※ ZIP 作成時は Python zipfile を使用（PowerShell は構造が壊れる）
※ 画像ファイル名は ASCII のみ使用（日本語名は Linux で問題を起こす）

■ 開発環境の切替（Windows ↔ Mac）
================================================================

1. GitHub からクローン:
   git clone https://github.com/bun3111/aoibase-hp.git

2. テーマディレクトリ: aoibase-hp/aoibase-theme/

3. デプロイ時のZIP作成:
   python3 -c "
   import zipfile, os
   theme_dir = 'aoibase-theme'
   with zipfile.ZipFile('aoibase-theme.zip', 'w', zipfile.ZIP_DEFLATED) as zf:
       for root, dirs, files in os.walk(theme_dir):
           for f in files:
               full = os.path.join(root, f)
               zf.write(full)
   "

4. CLAUDE.md / .claude/ ディレクトリはリポジトリ外
   → Mac側でも同様のCLAUDE.mdを設置する

■ 注意事項
================================================================

- Playwright MCP でWordPress管理画面操作可能（コンテンツ編集、画像アップロード等）

- CF7フォームはWordPress管理画面で設定済み（テーマファイルにはショートコードのみ）
- main.js はスライド数を動的取得するため、HTML側の増減だけでOK
